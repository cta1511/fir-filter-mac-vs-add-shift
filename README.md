# FIR Filter: MAC vs Add/Shift

This project compares FIR filter implementations using a MAC-based design and an add/shift-based design.

## Structure

- `Codes/` - Verilog source, testbench files, and MATLAB/Simulink-related test assets.
- `Designs/` - Block diagrams and Simulink design images.
- `Docs/` - Report and presentation documents.
- `Demos/` - Demo notes. Large video files are kept locally and are not tracked in Git.
- `Refs/` - Reference papers and documents used for the project.

## Notes

Large video files are intentionally ignored to keep the repository lightweight.
